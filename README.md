# CV Responsibilities Optimizer (Browser-Only)

A lightweight, 100% browser-based tool that analyzes multiple job descriptions and rewrites your current CV responsibilities into stronger, market-aligned bullet points. It also generates a “Core Competencies” section you can copy and paste directly into your CV.

## Features

* Paste 5–10 job descriptions (separate with `====`) to extract common requirements
* Paste your current responsibilities (supports `-`, `•`, `*`, `–`, and numbered lists)
* Generates:

  * Optimized Responsibilities (CV-ready bullets)
  * Core Competencies (CV-ready bullets)
  * Before vs After comparison
  * 10-Day learning plan for missing skills
* One-click Copy All + Download as text
* Saves progress locally (LocalStorage)
* No backend, no tracking, no data sent anywhere

## How to Run

Download or clone this repo, then open `index.html` in your browser.
